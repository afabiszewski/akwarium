# Project Guidance

This app should stay simple, quick, and static.

- Prefer plain HTML and CSS for the core site; Tailwind CSS is also acceptable when it keeps styling fast and simple.
- Avoid adding build tools, frameworks, or runtime dependencies unless there is a clear need.
- Keep pages easy to open locally and easy to deploy as static files.
- The production site is served from GitHub Pages at https://afabiszewski.github.io/akwarium/.
- All changes must be responsive and render correctly on phone screens.
- Favor small, readable changes over abstractions.
